# fuadmm.github.io

Personal website for academic and professional portfolio. This website showcases academic achievements, research work, laboratory studies, publications, and professional information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Professional Layout**: Clean and modern design with smooth animations
- **Comprehensive Sections**:
  - About Me
  - Academic Information (Education, Teaching, Awards)
  - Research Work (Research Areas, Current Projects)
  - Laboratory Studies (Techniques, Facilities, Collaborations)
  - Publications (Journal Articles, Conference Proceedings, Book Chapters)
  - Contact Information

## Structure

- `index.html` - Main HTML file with semantic structure
- `styles.css` - Professional CSS styling with responsive design
- `script.js` - JavaScript for interactive features and animations

## Customization

To personalize the website with your information:

1. **Personal Information**: Update the name and professional title in `index.html`
2. **Academic Details**: Fill in your education, teaching experience, and awards
3. **Research Content**: Add your research areas, projects, and descriptions
4. **Laboratory Information**: Update techniques, facilities, and collaborations
5. **Publications**: Add your published works with proper citations
6. **Contact Details**: Update email, office location, and professional profile links
7. **Colors**: Modify color scheme in the `:root` section of `styles.css`

## Deployment

This site is deployed automatically via GitHub Pages at: `https://fuadmm.github.io/`

Any push to the main branch will automatically update the live site.
